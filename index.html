Mimeograph

CoffeeScript lib for PDF OCR and text extraction

Description

mimeograph is a simple CoffeeScript library to extract text and create searchable pdf files, OCRing where necessary. Each individual step in the process is a separate coffee-resque job allowing for interesting scaling options.

System Requirements

  • poppler-utils (pdftotext)
  • libtiff
  • Leptonica
  • tesseract v3.01 (svn trunk)
  • ImageMagick

Install Development (OS X)

Installation is a three step process:

  1. install node
  2. install mimeograph dependencies
  3. install mimeograph

Node

Mimeograph is currently tested with Node v0.4.8.

  1. Download node v0.4.8.
  2. Untar/unzip the contents and CD into the root directory (node-v0.4.8).
  3. Review the installation instructions found in README.md
  4. run "./configure"
  5. run "make"
  6. run "make install"
  7. run "node -v" to verify the installation has been successful.

Mimeo Dependencies

This assumes that you have Homebrew installed and are using it for pacakge management on OS X. If you aren't already using it download it here. If you are already using Macport or Fink and are happy with them, just use these instructions as a guide.

  1. run "brew update"
  2. run "brew install poppler ghostscript imagemagick leptonica redis"
  3. run "brew info tesseract". At the current time the Homebrew recipe for tesseract was for v3.00 of tesseract. If the command you ran does not indicate >= v3.01 you will have to jump through some additional hoops.
    1. if Homebrew has a package for tesseract >= 3.01 run: "brew install tesseract"
    2. if Homebrew doesn't have a package for tesseract >= 3.01 run "brew edit tesseract" and update the contents of the tesseract.rb with this.

Mimeograph

  1. run "git clone git@github.com:morologous/mimeograph.git"
  2. run "cd mimeograph"
  3. run "make"

Running

Usage:
  mimeograph [OPTIONS] [ID] filename

  -h, --help         Displays options
  -v, --version      Shows certain's version.
  -w, --workers      Number of workers to create. (Default: 5)
  -s, --start        Starts a Mimeograph daemon.
  -p, --process      Kicks of the processing of a new file.
      --port         Redis port. (Default: Redis' default)
      --host         Redis host. (Default: Redis' default)

eg.  mimeograph -p test/test.pdf
     mimeograph -w 10 -s
